## 1. Dataset Folder Format

Create this folder structure:

```text
dataset/
  train/
    authentic/
    ai_generated/
    manipulated/
  val/
    authentic/
    ai_generated/
    manipulated/
  test/
    authentic/
    ai_generated/
    manipulated/
```

Put images into the correct folders.

- `authentic`: normal camera photos
- `ai_generated`: images generated by AI tools
- `manipulated`: edited, spliced, pasted, or heavily altered images

You can train without `manipulated` images for the first model. If `manipulated`
is empty, the trainer automatically skips it and trains a binary model:

```text
authentic vs ai_generated
```

Add `manipulated` images later when you are ready for the full 3-class model.

Start small if needed:

```text
train: 100+ images per class
val:   20+ images per class
test:  20+ images per class
```

More data gives better results.

## 2. Recommended Training: Google Colab

Use Colab if your laptop is low on RAM.

Steps:

1. Zip your dataset folder.
2. Upload it to Google Drive.
3. Open a Colab notebook.
4. Install dependencies:

```python
!pip install torch torchvision Pillow tqdm scikit-learn
```

5. Upload/copy this `backend/ml` folder into Colab.
6. Unzip your dataset so the path looks like:

```text
/content/dataset/train/authentic
/content/dataset/train/ai_generated
/content/dataset/train/manipulated
/content/dataset/val/authentic
/content/dataset/val/ai_generated
/content/dataset/val/manipulated
```

7. Run training:

```python
!python train_model.py --data /content/dataset --epochs 10 --batch-size 16
```

For your current dataset, this will train on:

```text
authentic
ai_generated
```

because the `manipulated` folders are currently empty.

8. Download the trained file:

```text
models/scanai_model.pt
```

9. Put it into your project here:

```text
D:\ScanAI\backend\ml\models\scanai_model.pt
```

10. Restart the backend.

## 3. Local Training Command

If you want to train locally:

```powershell
cd D:\ScanAI\backend\ml
..\venv\Scripts\python.exe train_model.py --data dataset --epochs 10 --batch-size 8
```

If you want to explicitly train only the current two classes:

```powershell
..\venv\Scripts\python.exe train_model.py --data dataset --classes authentic ai_generated --epochs 10 --batch-size 8
```

If you run out of memory, reduce batch size:

```powershell
..\venv\Scripts\python.exe train_model.py --data dataset --epochs 10 --batch-size 4
```

## 5. Important Notes

- The first training run may download EfficientNet weights.
- EfficientNet weights are cached inside `backend/ml/models/torch_cache`.
- Training needs many images. A tiny dataset will overfit.
- Keep a separate `test` folder and do not train on it.
- The backend still shows forensic checks even when the ML model is missing.
